# Todo API
## Pengenalan
Tugas Web Lanjutan. Membuat aplikasi todo list sederhana.

## Framework / Library

- Express Js.
- Sequelize untuk menggunakan teknik ORM dan database migration.
- JWT untuk autentikasi. 

## Database

- Postgresql versi 11.
